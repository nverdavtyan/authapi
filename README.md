## AuthApi

## Getting started

### Requirements

- Laravel 9 
- PostgreSQL - `config your db`
- NextJs - `npm install `
### Initial setup

1. Customize the `.env`
2. Run `composer install  php artisan migrate`

### Running AuthApi ! 

# backend
1. Backend `cd backend`
2. Run `php artisan serve`

# client
1. client `cd client`
2. Install `npm install`
3. Run `npm run dev`
